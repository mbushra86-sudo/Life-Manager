# مدير اليوم — تطبيق PWA

لوحة حيّة لتنظيم اليوم حول أوقات الصلاة (الكويت)، الدوام، القرآن، الدايت، العائلة، والنوم. يتثبّت على التلفون كتطبيق ويشتغل offline.

## الملفات
- `index.html` — التطبيق كامل
- `manifest.webmanifest` — تعريف التطبيق (الاسم، الأيقونة، يشتغل ملء الشاشة)
- `sw.js` — service worker (يخلّيه يشتغل بدون نت + يفعّل زرار التثبيت)
- `icon.svg` / `icon-192.png` / `icon-512.png` / `icon-maskable.png` — أيقونات التطبيق

## الرفع على GitHub Pages (خطوة بخطوة)

1. ادخل github.com واعمل repository جديد (مثلاً اسمه `day`)، خليه **Public**.
2. اضغط **Add file → Upload files**، واسحب كل الملفات اللي في الفولدر ده (بما فيها الأيقونات)، ثم **Commit**.
3. من الـ repo: **Settings → Pages**.
4. تحت **Source** اختار **Deploy from a branch**، والـ branch خليه `main` والمجلد `/ (root)`، ثم **Save**.
5. استنى دقيقة، هيظهرلك رابط بالشكل: `https://اسم-حسابك.github.io/day/`

## التثبيت على التلفون

1. افتح الرابط في **Chrome** (أندرويد) أو **Safari** (آيفون).
2. أندرويد: هيظهر زرار **"تثبيت التطبيق"** جوه التطبيق نفسه، أو من قائمة Chrome **Install app**.
3. آيفون: زرار المشاركة ⬆ → **Add to Home Screen**.
4. هتلاقي أيقونة التطبيق على الشاشة الرئيسية، تفتح ملء الشاشة وتشتغل offline.

## ملاحظات
- أوقات الصلاة تتحدّث تلقائياً من Aladhan API حسب توقيت الكويت، وآخر قيمة محفوظة تُستخدم بدون نت.
- كل البيانات (التاسكات، العادات، الأوقات) محفوظة محلياً على جهازك في المتصفح.
- زرار "تثبيت التطبيق" يظهر فقط على https — لذلك لازم يكون على رابط GitHub Pages، مش ملف محلي.
